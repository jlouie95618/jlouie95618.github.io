Notes:
    ID Syntax: id = "id_name" w/ #id_name {...}
    Class Syntax: class = "class_name" w/ .class_name {...}

    example Javascript function:
    var sayHello = function(name) {
        console.log('Hello ' + name);
    };

    To include CSS: use the link element (<link rel="stylesheet" href="stylesheet.css">)
    To include jQuery and Javascript: use the script element (<script src = "javascript_or_jquery_app.js"></script>)

Standard jQuery:

$(document).ready(function(){
    $("div").slideDown("slow")
});

$(document).ready(function() {
    $('div').mouseenter( function() {
        $('div').fadeTo("fast", 1);
    });
    
    $('div').mouseleave( function() {
        $('div').fadeTo('fast', 0.5);
    });
    
});

Structure of jQuery functions:

function(input1, input2, etc) {
    Do a thing
    Do another thing
    Do yet another thing!
}

Typical jQuery Event Setup:
    $(document).ready(function() {
        $('thingToTouch').event(function() {
            $('thingToAffect').effect();
        });
    });
